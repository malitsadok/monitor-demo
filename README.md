# Monitor Demo
Monitor Demo is a web app to monitor a specific local folder. 

The project leyout includes 2 node servers
* Backend monitor
* Client UI

## Prerequizites 
* node 12.13.1
* npm 6.12.1
* nodemon 2.0.1
> nodemon can be installed globaly or as dev-dependency (default)

## Runnig the project
clont this repo
#### Start backend
From root folder
```
cd  backend
npm install
npm run dev-server
```
#### Start UI
In root folder
```
npm install
npm start
```
